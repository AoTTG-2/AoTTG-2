Class FengGameManagerMKII

The Original AoTTG1 god class of 17k+ lines of code! This class used to do literally everything, and still does too much. Do not add new code to this class, and rather refactor the logic to a service or a different component. This class should eventually only have 2 responsibilities:

  1. Initializing the Settings
  2. Loading a level and the respective gamemode
