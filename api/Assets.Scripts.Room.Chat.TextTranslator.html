Translate(String, String, String, Action<String[]>)

A Unity coroutine that takes in a string, an ISO 639-1 code to translate from (or "auto"), an ISO 639-1 code to translate to, and a callback action.

Declaration
public static IEnumerator Translate(string text, string langCodeFrom, string langCodeTo, Action<string[]> callback)
Parameters
Type Name Description
System.String text

The original text to translate
System.String langCodeFrom

The original language to translate from (use "auto" to have Google auto-detect).
System.String langCodeTo

The language to translate the text to.
System.Action<System.String[]> callback

Gets invoked with a string[] array that returns either the language code and translated text, or the error Unity encountered.
Returns
Type Description
System.Collections.IEnumerator

Nothing, the results are invoked by the 'callback' parameter.
Examples

Example usage of Translate(String, String, String, Action<String[]>)

StartCoroutine(TextTranslator.GetTranslatedText("Guten tag!", "auto", "en", results => {
    if (results.Length > 1) // We have both a language code AND the translated text!
    {
        Console.WriteLine($"*From {results[0]}*: {results[1]");
    } else // Unity encountered an error somewhere
    {
        Console.WriteLine($"*Translation Error: {results[0]");
    }
}));
